Hexo笔记

2021-12-17
  • 笔记
  • Hexo

安装Hexo (Windows环境)

  1. 下载并安装Node.js和npm
  2. 安装npm：
    npm install hexo-cli -g

使用Hexo

新建网站：hexo init
新建内容：hexo new <Scaffold> <name>
生成静态网站文件：hexo g
清理静态网站文件：hexo clean
本地架临时网站服务：hexo s
一键布置：hexo deploy

升级Hexo

  1. 升级Node.js和npm
  2. 升级Hexo
    npm install hexo-cli -g
  3. 升级插件
    npm install npm-upgrade
    npm-upgrade

网络爬虫杂谈

2021-12-16
  • 杂谈
  • network

从隐私谈起

百度CEO李彦宏在谈到AI发展时候曾经说过一句非常著名的话：“我想中国人可以更加开放，对隐私问题没有那么敏感。如果他们愿意用隐私交换便捷性，很多情况下他们是愿意的，那我们就可以用数据做一些事情。”
且不论这句话对错与否，这句话的背景条件正是因为现在互联网公司实质掌握了用户的隐私资料。例如像淘宝的消费记录，百度的搜索记录，微信的聊天记录，高德的定位记录等等。当多方位的个人资料被公司们掌握时，就可以针对用户或者群体进行“画像”，可谓之现代科技版的福尔摩斯。而正如同福尔摩斯依靠这种见微知著的识人能力能够断案一样，互联网公司也会借助这些画像来盈利，常见的方式就是个性化广告推荐。
所以说，为什么互联网会出现十几年没有实际盈利都有投资者前赴后继的去填坑的现象，参考这篇关于视频网站盈利模式的文章，是因为表面免费的模式下，用户本身正是互联网公司所要出售的商品。因此，用户的信息自然是互联网公司最看重的“财产”。通常来说，一个网站的用户信息自然只有该网站的所有者能获取，但爬虫正是打破了这个常规的破局者。

总览

信息时代，能有一个自己的网站，总是一件十分酷炫的事。然而，即使是一个非常普通的个人博客站，其背后隐藏的各种技术仍然足够让普通人望而却步。就算是学习计算机相关专业的学生，在接触和学习相关的内容的时候也不免踩一些坑，无谓的浪费许多宝贵的时间。
所以今天在这里，本人打算从头到尾，给各位读者总结建站的整个原理，流程，以及一些个人花了很长时间踩过的各种坑。如果能够因此帮到一个两个有相关需求的朋友，那就能体现到这篇博文的价值了。
本篇文章主要面向对于计算机网络了解不多的新手，所以我会着重介绍概念，理论以及推荐相关的工具，网站，作指路之效。具体工具的使用步骤，一部分后续可能会另外更新专门的文章；即使没有，有了这些关键词之后在互联网上应该也很容易找到相关资料，请善用搜索引擎。

区段元素

链接

行内式

This is [an example](http://example.com/ "Title") inline link.

[This link](http://example.net/) has no title attribute.

参考式

This is [an example][id] reference-style link.

然后在文件任意处添加

[id]: http://example.com/  "Optional Title Here"

更多用法详见参考

强调

使用一个*或_

*斜体*
_italic_

使用两个*或_

**粗体**
__bold__

如果*或_两边都是空白，它们等同于普通符号
想要literally把文字用这两个符号框起来请使用转义符号\

代码

小段行内代码用`包起来

Java uses `System.out.print()` to print on the teminal

而在行内代码包含了`的情况下，则用多个`来包裹代码内容

``This is an exaple for ` in the code section``

当`出现在开头或结尾时，请使用空格

`` `this is sample` ``
